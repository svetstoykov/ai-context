# C# XML Docs

## What It Is

A reusable skill for writing and reviewing C# XML documentation comments on public APIs.

## When To Use

Use this when an agent is asked to add, fix, review, or generate C# XML comments, especially tags such as `<summary>`, `<remarks>`, `<param>`, `<returns>`, `<exception>`, `<see>`, and `<paramref>`.

## How To Use

Copy this folder into an agent skill directory, or reference `SKILL.md` directly from project instructions. Apply the skill before generating public C# APIs that require documentation and before reviewing documentation comments for correctness.

## Contents

- `SKILL.md` - agent-facing instructions for C# XML documentation style, tag selection, examples, and review checks.
