# Testing Principles

## What It Is

A reusable skill for writing, reviewing, and structuring reliable automated tests.

## When To Use

Use this when an agent is asked to write tests, add unit tests, review test code, design integration tests, diagnose brittle tests, or decide which test layer should cover a behavior.

## How To Use

Copy this folder into an agent skill directory, or reference `SKILL.md` directly from project instructions. Apply the skill before writing or reviewing tests so the test strategy matches the behavior, risk, and dependency boundaries involved.

## Contents

- `SKILL.md` - agent-facing testing principles, layer selection guidance, and review checklist.
